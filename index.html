What is this test?

Six AI models got the same homework: “write a program that opens a PyTorch file — the file where an AI stores all its numbers — using nothing but plain C++. No PyTorch, no Python, no libraries that do the work for you.”

Then their programs were fed files that get meaner and meaner. Tiers 1–5 test correctness: plain files, nested files, weird-but-honest files — did you read the numbers right? Tier 6 tests robustness: booby-trapped files that try to trick the program into running hidden code. Fall for it once and you're disqualified.

Finally, every model was handed the exact same score sheet (results.json) and told: build a dashboard. This page hangs their answers side by side — ranked by overall score (60% how well the code did, 40% how good it looks) — so you can judge not just who passed, but who explained it best.
